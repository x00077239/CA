Bash Script Tutorial for Year 4 IT Students

This is a bash script tutorial for a year 4 IT Student

A Bash script is a plain text file which contains a series of commands. These commands are a mixture of commands we would normally type ouselves on the command line (such as ls or cp)and commands we could type on the command line but generally wouldn't.

Top Commands used within BASH

ls — List directory contents

echo — Prints text to the terminal window

touch — Creates a file

mkdir — Create a directory

grep — search

man — Print manual or get help for a command

pwd — Print working directory

cd — Change directory

mv — Move or rename directory

rmdir — Remove directory

locate — Locate a specific file or directory

less — view the contents of a text file

compgen — Shows all available commands, aliases, and functions

cat — Read a file, create a file, and concatenate files

| — Pipe

head — Read the start of a file

tail — Read the end of a file

chmod — Sets the file permissions flag on a file or folder

exit — Exit out of a directory

Creating your own bash commands

Custom commands in Bash are known as “aliases”. Aliases are an abbreviation, or a means to avoid typing a long command sequence. They can save a great deal of typing at the command line so you can avoid having to remember complex combinations of commands and options.

Example of using aliases

Syntax: alias alias_name = “command_to_run” A very simple example would look like this: alias c = “clear” Now every time you want to clear the screen, instead of typing in clear, you can just type c and you’ll be good to go. Using Bash Commands to display system analysis

lspci - this will list the PCI devices within the system

lsusb - this will list the USB devices within the system

lsmod - this will list the drivers within the system

To continously display System Log

To continually display the system log you should use this command: tail ­f /var/log/syslog

File Properties

Using the statmyfile command will list the name, size, type, permissions, owner, time stamps info on file system containing statmyfile

Disk tools

using df will display disk space and usage for all drives

using mount will display all mounted devices

using sync will write all pending operations to disk Very important to do before retreiving a USB stick or a memory card!

Time Processes

to execute a program every 10 seconds you would use the following command watch ­n 10 myprogram
